# クイックスタートガイド

RapidWireを使って、独自の通貨経済圏を作り上げるためのステップバイステップガイドです。

## シナリオ
あなたは「GamerCoin (GMC)」という新しい通貨を作り、友人に配布し、さらに誰でも交換できるように市場（DEX）を作りたいと考えています。

---

## ステップ1: 通貨を作る

まず、あなただけの通貨を発行しましょう。

コマンド:
```
/currency create name:GamerCoin symbol:GMC supply:1000000 hourly_interest_rate:0.001
```

- **name**: GamerCoin
- **symbol**: GMC
- **supply**: 1,000,000 (100万枚)
- **hourly_interest_rate**: 0.001% (ステーキングした時の金利)

成功すると、あなたのウォレットに100万GMCが入ります。
`/balance` コマンドで確認してみましょう。

---

## ステップ2: 友人に配る

経済を回すには参加者が必要です。友人にGMCを送ってみましょう。

コマンド:
```
/transfer user:@Friend amount:1000 symbol:GMC
```

これで友人は1,000 GMCを受け取りました。

---

## ステップ3: 市場（流動性プール）を作る

GMCだけを持っていても、他の通貨と交換できなければ不便です。
サーバーには既に「StandardCoin (STD)」という基軸通貨が存在し、みんなが持っていると仮定します。
GMCとSTDを交換できるプールを作りましょう。

プールを作るには、両方の通貨を同じ価値分だけペアで提供する必要があります。
ここでは、`10,000 GMC` と `100 STD` の価値が等しいと定義してプールを作ります（つまり、1 STD = 100 GMC のレート）。

コマンド:
```
/lp create symbol_a:GMC amount_a:10000 symbol_b:STD amount_b:100
```

これで、誰でもGMCとSTDを交換できる市場が完成しました！

---

## ステップ4: スワップ（交換）を試す

試しに、手持ちのSTDを使ってGMCを買ってみましょう（またはその逆）。

コマンド:
```
/swap from_symbol:STD to_symbol:GMC amount:10
```

10 STDを支払って、現在のレートに基づいたGMCを受け取ることができます。
先ほど自分でプールを作ったので、あなたがこの取引の手数料を受け取ることになります（手数料はプールに蓄積され、プールを引き出すときに受け取れます）。

---

## ステップ5: ステーキングで増やす

余ったGMCはステーキングして増やしましょう。

コマンド:
```
/stake deposit amount:50000 symbol:GMC
```

5万GMCを預けました。設定した時利 `0.001%` に従って、毎時間少しずつ残高が増えていきます。
いつでも `/stake withdraw` で引き出すことができます。

---

## 次のステップ

これで基本的な経済圏が完成しました。さらに発展させるには…

- **スマートコントラクト**: 「GMCを払うと特別なロールが貰える」自動販売機を作る。
- **請求書**: 「イベント参加費」としてGMCを参加者に請求する。
- **Renounce**: ある程度普及したら、Mint権限を放棄してGMCを「みんなのもの」にする。
